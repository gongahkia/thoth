## Description

Thoth is a compact XCOM-lite tactics game built in LOVE/Lua. Lead six auditors through a hostile Buried Archive route, inspect intent traces, bend cover and LoS, declare overwatch, and keep the route machine intact while every board-loaded action resolves deterministically.

This Phase 6 alpha contains the current tactical vertical slice:

- One Buried Archive route with visible rewards, complications, and boss-gate pressure.
- Six procedural tactical board variants.
- Six-class starting squad: Warden, Duelist, Apothecary, Thief, Arcanist, Lamplighter.
- Objective families for entry audit, shelf protection, proof extraction, ledger repair, sealed shortcut, and Vault Regent final.
- Fog-of-war, cover/flank reads, LoS previews, overwatch cones, hidden/revealed intent, intent legend, and tile inspector.
- One selected elite, Shelf Knight, and one selected boss, Vault Regent.
- Fixed-seed procgen validator and deterministic replay checks.
- Accessibility toggles for colorblind modes, high contrast, subtitles, font scale, reduced motion, and tactical overlay tuning.

## Controls

- Cursor / aim tile: `WASD`, D-pad, or left stick.
- Select / commit: mouse left click, `Enter`, or controller `A`.
- Inspect preview: `Space` or controller `X`.
- Rotate view: `[` / `]` or controller shoulders.
- End turn: `E`.
- Pause/back: `Escape` or controller back.
- Settings: available from title and pause.
