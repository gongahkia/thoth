# Phase 7.2 Combat Juice Pass

Date: 2026-06-20

Implemented:

- Hit pause after structured combat impact events.
- Toggleable screen shake via Settings -> Screen Shake.
- Damage numbers for HP/stress impacts.
- Crit feedback for marked direct hits.

Scope:

- Combat impact metadata is emitted by hero and enemy skill resolution.
- Reduced Motion suppresses hit pause and shake.
- Screen Shake disables cutscene shake without disabling damage numbers.
