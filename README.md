# Thoth

Pseudo-3D terrain exploration prototype in LÖVE. The focus is deterministic terrain generation: plates, uplift, rainfall, drainage, erosion, rivers, and biomes.

## Development

```sh
make test
make run
```

Controls:

- `WASD`: walk / strafe
- `Shift`: sprint
- mouse or `Q` / `E`: look
- arrow up/down: pitch
- `F`: toggle mouse look
- `R`: new seed
