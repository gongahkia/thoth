# Thoth Music Assets

Music is driven by `tracks.lua`.

Runtime contract:

- File format: MP3 or OGG stream sources
- Default path root: `assets/music/`
- Missing tracks are skipped at runtime
- Download license certificates before committing raw music files

Planned filenames:

